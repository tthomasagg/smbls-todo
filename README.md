# Symbols MCP Test - To-Do App

A simple To-Do application built with [Symbols](https://www.symbols.dev/).

## Features

- Create, edit, and delete tasks
- Sort tasks by different criteria
- Clean component-based architecture

## Project Structure

```
smbls/
├── components/    # UI components (TaskForm, SortControls)
├── functions/     # Business logic (taskService, taskSorter, taskRepository)
├── pages/         # Page components (todo)
└── types/         # Type definitions (Task)
```

## Getting Started

### Prerequisites

- Node.js

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

### Build

```bash
npm run build
```

## Tech Stack

- [Symbols](https://www.symbols.dev/) - UI framework
- [Parcel](https://parceljs.org/) - Build tool
